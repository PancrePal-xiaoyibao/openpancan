# OpenPanCan Gene Base 膵臓癌遺伝子データベース

**膵臓癌ゲノミクスのための AI 駆動型インサイトプラットフォーム**

---

## 概要

OpenPanCan Gene Base は、膵臓癌研究のための AI 駆動型ゲノム解析を支える包括的なオープンソース遺伝子データベースです。複数の信頼できるデータソースから遺伝子レベルの情報を集約・整理・構造化し、機械学習モデル、臨床意思決定支援ツール、研究アプリケーションに統一された基盤を提供します。

膵臓癌は依然として最も致死率の高い悪性腫瘍の一つであり、5 年生存率は 12% 未満です。腫瘍形成を促進し治療反応に影響を与える変異、シグナル伝達経路、バイオマーカーを含むゲノムの全容を理解することは、治療成績の向上に不可欠です。OpenPanCan Gene Base は、高品質な遺伝子データをアクセス可能かつ相互運用可能で AI 対応にすることで、この理解を加速します。

## 主要な膵臓癌関連遺伝子

本データベースは、膵管腺癌（PDAC）において臨床的・生物学的に最も重要な以下の遺伝子に焦点を当てています：

| 遺伝子 | 膵臓癌における役割 |
|--------|-------------------|
| **KRAS** | 癌遺伝子。90% 以上の PDAC 症例で活性化変異が検出され、膵臓腫瘍形成の主要な駆動因子。 |
| **TP53** | 腫瘍抑制遺伝子。約 75% の PDAC で不活性化。ゲノム不安定性と無秩序な増殖を引き起こす。 |
| **SMAD4** | 腫瘍抑制遺伝子。約 55% の PDAC で欠失または変異。転移拡散と予後不良に関連。 |
| **CDKN2A** | 腫瘍抑制遺伝子（p16）。90% 以上の PDAC で不活性化。細胞周期制御を破綻させる。 |
| **BRCA1** | DNA 修復遺伝子。生殖細胞系列変異は PDAC リスクを上昇させ、PARP 阻害剤治療の予測バイオマーカー。 |
| **BRCA2** | DNA 修復遺伝子。最も一般的な遺伝性 PDAC 感受性遺伝子。プラチナ製剤化学療法と PARP 阻害剤の使用指針。 |
| **PALB2** | Fanconi 貧血経路の DNA 修復遺伝子。変異は PDAC リスク増加と PARP 阻害剤感受性を予測。 |
| **ATM** | DNA 損傷応答遺伝子。家族性 PDAC の約 5% に変異。潜在的な治療標的。 |
| **ARID1A** | クロマチンリモデリング遺伝子（SWI/SNF 複合体）。PDAC の約 8% で変異。新興治療標的。 |
| **RNF43** | Wnt 経路制御因子。PDAC の約 6% で変異、特に膵胆管サブタイプで。 |
| **STK11** | 腫瘍抑制遺伝子（LKB1）。稀だが臨床的に重要な変異。KRAS 駆動癌と関連。 |
| **TGFBR2** | TGF-β シグナル受容体。マイクロサテライト不安定性を伴う PDAC の約 5% で変異。 |
| **MAP2K4** | ストレス活性化キナーゼ。PDAC で再発性変異。JNK シグナル伝達に関与。 |
| **GNAS** | G タンパク質シグナル分子。IPMN（管内乳頭粘液性腫瘍）に特徴的な変異。 |
| **KDM6A** | ヒストン脱メチル化酵素。PDAC の約 5% で変異。性別による影響を持つエピジェネティック制御因子。 |
| **RBM10** | RNA 結合タンパク質。PDAC で再発性変異。選択的スプライシング制御に関与。 |
| **SF3B1** | スプライセオソーム構成因子。PDAC で変異が検出され、異常なスプライシングパターンと関連。 |

## データソース

| ソース | 説明 |
|--------|------|
| **COSMIC** | 世界最大のがん体細胞変異データベース。変異頻度、機能的影響予測、薬剤耐性データを提供。 |
| **TCGA-PAAD** | 数百例の全エクソームシーケンス、RNA-seq、メチル化、臨床データを含む膵臓腺癌のマルチオミクス解析。 |
| **OncoKB** | MSKCC が整備する発がん性変異のナレッジベース。FDA 承認薬および治験薬情報を含む。 |
| **ClinVar** | NCBI のヒト遺伝子変異公開アーカイブ。生殖細胞系列がん感受性変異の臨床的解釈を含む。 |
| **GTEx** | 正常ヒト組織の遺伝子発現と eQTL データ。膵臓組織のベースライン発現コンテキストを提供。 |

## OpenRare との関係

OpenPanCan Gene Base は **OpenRare**（希少疾患ゲノム解析のためのオープンソースプラットフォーム）から多大な影響を受けています。OpenRare コミュニティの先駆的な取り組みに感謝の意を表します。OpenRare が希少疾患全般を対象とするのに対し、本プロジェクトは膵臓癌に特化し、PDAC 研究固有の要件に合わせたデータモデルと AI 統合を提供します。
