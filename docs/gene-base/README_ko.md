# OpenPanCan Gene Base 췌장암 유전자 데이터베이스

**췌장암 유전체학을 위한 AI 기반 인사이트 플랫폼**

---

## 개요

OpenPanCan Gene Base는 췌장암 연구를 위한 AI 기반 유전체 분석을 지원하는 포괄적인 오픈소스 유전자 데이터베이스입니다. 여러 권위 있는 데이터 소스에서 유전자 수준의 정보를 집계, 큐레이션, 구조화하여 머신러닝 모델, 임상 의사결정 지원 도구 및 연구 애플리케이션을 위한 통합 기반을 제공합니다.

췌장암은 여전히 가장 치명적인 악성 종양 중 하나로, 5년 생존율이 12% 미만입니다. 종양 형성을 유발하고 치료 반응에 영향을 미치는 돌연변이, 신호 경로, 바이오마커 등 유전체적 특성을 이해하는 것이 환자 예후 개선에 매우 중요합니다. OpenPanCan Gene Base는 고품질 유전자 데이터를 접근 가능하고 상호 운용 가능하며 AI에 최적화된 형태로 제공함으로써 이러한 이해를 가속화하는 것을 목표로 합니다.

## 주요 췌장암 유전자

본 데이터베이스는 췌관 선암종(PDAC)에서 임상적·생물학적으로 가장 중요한 유전자들을 중심으로 합니다:

| 유전자 | 췌장암에서의 역할 |
|--------|------------------|
| **KRAS** | 종양유전자. PDAC 사례의 90% 이상에서 활성화 돌연변이 발견. 췌장 종양 형성의 주요 동인. |
| **TP53** | 종양억제유전자. PDAC의 약 75%에서 불활성화. 유전체 불안정성과 통제되지 않은 증식을 유발. |
| **SMAD4** | 종양억제유전자. PDAC의 약 55%에서 결실 또는 돌연변이. 전이 확산 및 불량한 예후와 관련. |
| **CDKN2A** | 종양억제유전자(p16). PDAC의 90% 이상에서 불활성화되어 세포 주기 조절을 방해. |
| **BRCA1** | DNA 복구 유전자. 생식세포 돌연변이는 PDAC 위험을 증가시키며 PARP 억제제 치료의 예측 바이오마커. |
| **BRCA2** | DNA 복구 유전자. 가장 흔한 유전성 PDAC 감수성 유전자. 백금 기반 화학요법과 PARP 억제제 사용 지침 제공. |
| **PALB2** | 판코니 빈혈 경로 DNA 복구 유전자. 돌연변이는 PDAC 위험 증가 및 PARP 억제제 민감성 예측. |
| **ATM** | DNA 손상 반응 유전자. 가족성 PDAC의 약 5%에서 돌연변이. 잠재적 치료 표적. |
| **ARID1A** | 염색질 리모델링 유전자(SWI/SNF 복합체). PDAC의 약 8%에서 돌연변이. 신흥 치료 표적. |
| **RNF43** | Wnt 경로 조절자. PDAC의 약 6%에서 돌연변이, 특히 췌담관 아형에서. |
| **STK11** | 종양억제유전자(LKB1). 드물지만 임상적으로 중요한 돌연변이로 KRAS 구동 암과 관련. |
| **TGFBR2** | TGF-β 신호 수용체. 현미부수체 불안정성을 동반한 PDAC의 약 5%에서 돌연변이. |
| **MAP2K4** | 스트레스 활성화 키나제. PDAC에서 반복적 돌연변이. JNK 신호 전달에 관여. |
| **GNAS** | G-단백질 신호 분자. 췌관 내 유두상 점액 종양(IPMN)의 특징적 돌연변이. |
| **KDM6A** | 히스톤 탈메틸화효소. PDAC의 약 5%에서 돌연변이. 성별 편향 효과를 가진 후성유전 조절자. |
| **RBM10** | RNA 결합 단백질. PDAC에서 반복적 돌연변이. 선택적 스플라이싱 조절에 관여. |
| **SF3B1** | 스플라이세오솜 구성요소. PDAC에서 돌연변이 검출, 비정상적 스플라이싱 패턴과 관련. |

## 데이터 출처

| 출처 | 설명 |
|------|------|
| **COSMIC** | 세계 최대의 암 체세포 돌연변이 데이터베이스. 돌연변이 빈도, 기능적 영향 예측, 약물 내성 데이터 제공. |
| **TCGA-PAAD** | 수백 명 환자의 전체 엑솜 시퀀싱, RNA-seq, 메틸화 및 임상 데이터를 포함한 췌장 선암종 다중 오믹스 특성 분석. |
| **OncoKB** | MSKCC가 큐레이션한 종양유전 변이 지식 베이스. FDA 승인 약물 및 연구 중인 치료법 정보 포함. |
| **ClinVar** | NCBI의 인간 유전 변이 공개 아카이브. 생식세포 암 감수성 변이의 임상적 해석 포함. |
| **GTEx** | 정상 인체 조직의 유전자 발현 및 eQTL 데이터. 췌장 조직의 기준 발현 컨텍스트 제공. |

## OpenRare와의 관계

OpenPanCan Gene Base는 희귀 질환 유전체 분석을 위한 오픈소스 플랫폼인 **OpenRare**에서 큰 영감을 받았습니다. OpenRare 커뮤니티의 선구적인 노력에 감사드립니다. OpenRare가 희귀 질환 전반을 다루는 반면, 본 프로젝트는 췌장암에 특화되어 PDAC 연구의 고유한 요구사항에 맞춘 데이터 모델과 AI 통합을 제공합니다.
